Базовая часть задания реализована в mycode.cpp. Там же написана функция, 
	запускающая дополнительные преобразования. Дополнительная часть 
	реализована в dop.cpp.

Многоклассовая классификация:
	работает как на бинарной выборке, так и на мультиклассе.

Пирамида дескрипторов:
	составлена пирамида дескрипторов. Изображение поделено на 4 части.
	Для каждой части составлен вектор признаков, который конкатенировался 
	с векотором, составленным для всего изображения.

Нелинейные ядра SVM:
	к каждой компоненте вектора применено преобразование с константой 
	lambda = L_CONST*n, где L_CONST = 0.5 и n = -1,0,1. Сначала записана
	действительная, потом мнимая часть вычисления для каждой компоненты.
	Применяется после вычисления пирамиды дескрипторов.

Цветовые признаки:
	для изображения посчитаны цветовые признаки. Изображение разделено на
	64 клетки. Полученный вектор цветовых признаков конкатенирован с 
	вектором, полученным после применения нелинейного ядра SVM. (Вектор
	цветовых признаков записан в конец).

В файле ./run.sh лежит скрипт, запускающий обучение и классификатор 
	сначала для базовой части, потом для мультикласса.
